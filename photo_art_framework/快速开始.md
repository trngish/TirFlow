# 写真/艺术照训练框架 - 快速开始

## 1. 准备工作

### 1.1 下载基础模型
推荐下载写实类SDXL模型：
- SDXL 1.0 Base
- Realistic Vision SDXL
- 写真专用模型

放置到: `d:\TirFlow\models\`

### 1.2 准备训练数据
可以使用数据预处理功能自动处理，或手动准备：
- 写真照片（建议10-30张）
- 每张照片对应的.txt标签文件

目录结构：
```
dataset/
  001.jpg
  001.txt
  002.jpg
  002.txt
  ...
```

## 2. 快速启动

双击运行：
```bash
启动框架.bat
```

或命令行：
```bash
python main.py
```

访问: http://127.0.0.1:7860

## 3. 使用流程

### 3.1 数据预处理
1. 切换到「🖼️ 数据预处理」Tab
2. 设置输入/输出目录
3. 配置目标分辨率、触发词
4. 点击「开始预处理」

### 3.2 训练模型
1. 切换到「🚀 训练」Tab
2. 选择基础模型
3. 设置触发词和训练数据目录
4. 配置参数（LoRA维度、步数等）
5. 点击「开始训练」
6. 观察Loss曲线

### 3.3 生成图片
1. 切换到「🎨 图片生成」Tab
2. 选择基础模型和LoRA
3. 编写提示词
4. 点击「生成图片」

## 4. 推荐参数

### 训练参数（12GB显存）
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 分辨率 | 512×768 | 竖版写真比例 |
| LoRA维度 | 32 | 越大越精细 |
| 训练步数 | 3000 | 越多越精细 |
| Batch Size | 1 | 显存不足改1 |
| 学习率 | 1e-4 | 默认即可 |

### 生成参数
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 步数 | 25-35 | 越多越精细 |
| CFG | 7-8 | 提示词影响力 |
| 分辨率 | 512×768 | 可选更高 |

### 训练时间
- 3000步: 约3-4小时
- 4000步: 约4-5小时

## 5. 常见问题

Q: 显存不足？
A: 降低分辨率到512×512，Batch Size改为1

Q: 训练太慢？
A: 减少步数到2000，分辨率512×512

Q: 质量不好？
A: 增加训练步数到4000+，使用更高质量的数据
