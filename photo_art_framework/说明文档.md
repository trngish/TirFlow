# 写真/艺术照训练框架

基于 [kohya-ss/sd-scripts](https://github.com/kohya-ss/sd-scripts) 构建的一体化写真/艺术照 LoRA 模型训练解决方案。

## 功能模块

| 模块 | 说明 |
|------|------|
| **数据预处理** | 批量图片处理：居中裁剪、缩放、标签生成 |
| **LoRA训练** | SDXL LoRA 模型训练，实时 loss 曲线监控 |
| **图片生成** | SDXL + LoRA 写真图片生成，通过 Gradio 网页界面操作 |

## 项目结构

```
photo_art_framework/
├── config.py           # 配置与路径定义
├── train_engine.py     # 训练引擎（子进程封装）
├── preprocess.py       # 图片预处理（裁剪、缩放、生成标签）
├── generate.py         # SDXL LoRA 图片生成
├── ui.py               # Gradio 网页界面
├── main.py             # 程序入口
├── presets.py          # 训练预设管理
├── logging_config.py   # 日志配置
├── presets/            # 训练预设定义 (.toml)
│   ├── portrait_photo.toml    # 写真风格
│   ├── character_portrait.toml # 人物肖像
│   ├── style_realistic.toml   # 写实风格
│   └── ...
└── output/             # 训练输出（模型、日志、样本图）
```

## 使用流程

### 1. 数据预处理

- 选择原始图片目录（支持递归扫描子文件夹）
- 设置目标分辨率（512/768/896/1024）
- 选择保持原图比例 或 裁剪为正方形
- 设置触发词和重复次数
- 点击"开始预处理"生成训练数据

### 2. LoRA 训练

- 选择基础模型（SDXL）
- 选择预设或手动调整参数
- 点击"开始训练"
- 实时查看训练进度和 Loss 曲线
- 支持中断后继续训练

### 3. 图片生成

- 选择基础模型
- 选择已训练的 LoRA 模型
- 填写提示词（支持中英文）
- 设置分辨率、步数、CFG、Seed
- 点击"生成图片"

## 训练参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Network Dim | 32 | LoRA 秩维度，越大越精细但训练更慢 |
| 训练步数 | 3000 | 最大训练步数 |
| Batch Size | 1 | 批大小，显存不足时设为 1 |
| 学习率 | 1e-4 | AdamW8bit 优化器学习率 |
| 分辨率 | 512x768 | 竖版写真比例 |
| Gradient Accumulation | 4 | 梯度累积步数 |

## 常见问题

**Q: 训练中断后如何继续？**

A: 勾选"启用续训练"，选择对应的 `-state` 目录下的 safetensors 文件即可从断点继续。

**Q: 显存不足怎么办？**

A: 将 Batch Size 设为 1，减少 resolution，或关闭 gradient_checkpointing。

**Q: 生成图片质量不佳？**

A: 尝试增加训练步数，调整 LoRA 权重（0.8-1.2），或更换更好的基础模型。
